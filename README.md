# MixClaw

A lightweight, full-featured AI Agent framework written in Go, combining the best of PicoClaw (extreme light weight, speed) and OpenClaw (rich UI, features).

## Features
- **Extremely Lightweight**: Built in Go, runs in a single binary with < 30MB RAM usage.
- **Built-in Web UI**: A modern web interface embedded directly into the Go binary (`go:embed`). No Node.js required.
- **Multi-Agent Swarm**: Supports spawning Sub-agents for complex tasks.
- **Rich Skills System**: Add and manage custom skills easily via the Web UI.
